Why duckmapR

Color that works as hard
as your science

Every palette in duckmapR is built from the Lab color space up, optimized with CIEDE2000 perceptual distance so equal data steps produce equal visual steps — no false highlights, no vanishing detail.

Lab-space design

Color maps are designed as floating-point Lab paths, not by hand-picking hex codes. CIEDE2000 resampling ensures lightness advances uniformly from end to end.

CVD-safe by construction

All color maps are universally readable by people with and without color vision deficiency, validated for deuteranopia, protanopia, and tritanopia. Colors and transitions and between them maintain distinctiveness across maps.

Bilateral diverging symmetry

Diverging color maps are optimized for scientific data visualization, with balanced left/right arm transitions, paired cumulative symmetry, and symmetrical interval pacing. True-white midpoints mean zero reads as zero.

Grayscale readable

Monotone lightness throughout. Every sequential and diverging color map converts gracefully to grayscale — essential for print, accessibility, and OLED dark-mode displays.

Built-in ggplot2 support

Drop-in scale_fill_duckmap() and scale_color_duckmap() functions for continuous, discrete, and categorical data.

API reference

  
Five functions


  

    
      

        Function
        Purpose
        Key arguments
      

    
    
      

        duckmap()
        Return a character vector of n hex colors interpolated from the requested palette. Type is resolved from the palette name by default.
        palette name · type "all" / "seq" / "div" / "cat" · n integer · reverse · style "standard" / "vivid" · colors
      

      

        duckmap_summary()
        List color maps and draw a visual summary strip for all color maps of the selected type.
        type · plot logical · style "standard" / "vivid"
      

      

        duckmap_info()
        Return structured color map metadata (name, type, default flag, default n, available styles) as a data frame.
        type "all" / "seq" / "div" / "cat"
      

      

        scale_fill_duckmap()
        ggplot2 fill scale for continuous, discrete, or categorical data.
        palette · type · discrete · reverse · style · colors · passes ... to ggplot2
      

      

        scale_color_duckmap()
        ggplot2 color (colour) scale — same interface as scale_fill_duckmap. scale_colour_duckmap is an alias.
        palette · type · discrete · reverse · style · colors · passes ... to ggplot2
      

    
  







  
Installation

  
Get started in one line

  
Requires R ≥ 4.1.0. ggplot2 is imported — no extra setup needed for scales.


  

    

      
From GitHub (recommended)

      
# install.packages("pak")
pak::pak("travismallard/duckmapR")

    

    

      
From local source tarball

      
install.packages(
  "duckmapR_0.1.0.tar.gz",
  repos = NULL,
  type = "source"
)
